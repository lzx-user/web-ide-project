# Web IDE 开发文档 - T-02: 核心骨架与布局设计

## 🎯 核心目标

利用 Tailwind CSS 的 Flexbox 弹性盒子模型，搭建出经典的 IDE 布局：左侧固定宽度的侧边栏 + 右侧自适应的工作区。

## 🛠️ 开发思路与关键代码

### 1. 宏观布局策略 (Flex)

摒弃传统的 `float` 或百分比计算，采用现代前端标准的 `flex` 布局：

- **最外层**：`h-screen w-screen flex`（撑满全屏，内部元素横向排列）。
- **左侧边栏**：`w-[250px]` 固定宽度。
- **右侧工作区**：`flex-1`（自动吃掉剩余的所有屏幕宽度）。

### 2. 右侧垂直切割

右侧工作区继续使用 `flex-col` 进行垂直排列切割：

- **顶部导航栏**：`h-12` (固定高度 48px)。
- **底部终端**：`h-64` (固定高度 256px)。
- **中间编辑器区**：再次使用 `flex-1`，自动填满顶部和底部之间的剩余高度。

### 3. 布局雏形代码参考

```react
function App() {
  return (
    <div className="h-screen w-screen flex bg-gray-900 text-white font-sans">
      {/* 左侧侧边栏 */}
      <div className="w-[250px] bg-gray-800 border-r border-gray-700">
        <div className="p-4 border-b border-gray-700 font-bold">📂 资源管理器</div>
      </div>

      {/* 右侧主工作区 */}
      <div className="flex-1 flex flex-col">
        {/* 顶部 */}
        <div className="h-12 bg-gray-800 border-b border-gray-700"></div>
        {/* 中间自适应 */}
        <div className="flex-1"></div>
        {/* 底部终端 */}
        <div className="h-64 bg-[#0d1117] border-t border-gray-700"></div>
      </div>
    </div>
  );
}
```
