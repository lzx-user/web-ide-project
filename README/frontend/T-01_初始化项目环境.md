# Web IDE 开发文档 - T-01: 项目初始化与环境清理

## 🎯 核心目标
搭建基于 Vite + React 的前端脚手架，接入最新的 Tailwind CSS v4，并清理官方默认样式，为后续开发准备一张“干净的白纸”。

## 🛠️ 开发思路与关键步骤

### 1. 接入 Tailwind CSS v4
与 V3 版本繁琐的 `tailwind.config.js` 不同，V4 版本的接入极其精简，采用 Vite 插件和 CSS 原生 `@import` 机制。
- 安装依赖：`npm install @tailwindcss/vite`
- 修改 `vite.config.js` 注入插件：
  ```react
  import { defineConfig } from 'vite'
  import react from '@vitejs/plugin-react'
  import tailwindcss from '@tailwindcss/vite' // 引入插件
  
  export default defineConfig({
    plugins: [react(), tailwindcss()],
  })

### 2.清理默认样式 (大扫除)

- **清空全局 CSS**：删除 `src/App.css`，并将 `src/index.css` 内容替换为极简的 Tailwind 引入指令：

```react
@import "tailwindcss";
```

- **重置主入口**：将 `src/App.jsx` 的默认模板代码删除，编写一个全屏的深色测试容器，验证 Tailwind 是否生效。

```react
function App() {
  return (
    <div className="h-screen w-screen bg-gray-900 text-white flex items-center justify-center">
      <h1 className="text-4xl text-blue-400">环境准备就绪</h1>
    </div>
  )
}
export default App
```
