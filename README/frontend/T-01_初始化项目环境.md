# T-01：搭建全屏骨架与基础布局

## 1、开发思路

### **第一步：圈地建房（初始化环境）**

在盖房子之前，得先有一块平整的土地。我们要利用 Vite 快速生成 React 模板，并引入外企最主流的样式库 **Tailwind CSS**。

#### 1.1 创建项目

打开你的工作目录（比如桌面），运行：

```react
npm create vite@latest web-ide-project -- --template react
cd web-ide-project
npm install
```

#### 1.2 引入 Tailwind CSS（外企标配）

外企非常喜欢 Tailwind，因为它能让你的 HTML 看起来逻辑清晰且响应式极快。

```react
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### **第二步：画出“大客厅”和“走廊”（全屏 Flex 布局）**

我们要让 IDE 像真正的 VS Code 一样，无论窗口多大，都能**撑满全屏**且不出现莫名其妙的滚动条。

#### 2.1 核心代码逻辑

在 `App.jsx` 中，我们要利用 `flex` 布局把页面切成左右两块。

```react
function App() {
  return (
    // 1. 最外层容器：
    // h-screen w-screen 确保它“吃满”整个屏幕
    // flex 确保里面的东西是横着排的
    // bg-gray-900 给人一种深夜编程的高级暗色感
    <div className="h-screen w-screen flex bg-gray-900 text-white font-sans">
      
      {/* 2. 左侧：预留给 Sidebar 的位置 (固定宽度) */}
      <div className="w-[250px] bg-gray-800 border-r border-gray-700">
        {/* 这里未来放 Sidebar */}
      </div>

      {/* 3. 右侧：主工作区 (flex-1 占据剩余所有空间) */}
      <div className="flex-1 flex flex-col">
          {/* 这里未来放 Header、Editor 和 Terminal */}
          <div className="flex-1 bg-gray-900">编辑器预览区域</div>
          <div className="h-64 bg-black">底部终端预览区域</div>
      </div>

    </div>
  );
}
```

### **第三步：验证骨架是否稳固**

此时你需要启动项目，看看屏幕是否被完美分割：

1. **终端输入**：`npm run dev`。
2. **检查点**：
   - 左边是否有条深色的竖线（Border-r）？
   - 调整浏览器窗口大小时，右侧区域是否会自动拉伸？
   - 背景色是否呈现出深邃的灰色 `bg-gray-900`？
